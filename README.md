# Smalltricks

A tactical hex-based strategy game.

**Play now:** https://pakoito.github.io/smalltx/

## About

Smalltricks is a two-player hex-grid wargame where players maneuver units to damage their opponent's castle while defending their own.

### Features
- 6x6 hex grid battlefield
- Multiple unit types with unique abilities
- Strategic movement and combat phases
- Draft mode or preset armies
- Clean, responsive UI
